# 1. 互斥锁 

互斥锁(Mutex) 是保证线程安全访问共享资源的一种常见机制。标准库提供了 `std::sync::Mutex` ，它用来在多线程环境下保护数据的安全。

```rust
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    // Arc 原子引用计数，在多个线程共享所有权
    let counter = Arc::new(Mutex::new(0));
    let mut handlers = vec![];

    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handler = thread::spawn(move || {
            // 每个线程通过 lock() 获取互斥锁
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handlers.push(handler);
    }

    for handler in handlers {
        // 等待所有线程完成
        handler.join().unwrap();
    }

    println!("Result: {}", *counter.lock().unwrap());
}
```



# 2. 原子操作

标准库中原子类型 (`AtomicUsize`等)，允许在共享数据上的原子操作，确保这些操作在并发环境中的安全性和效率

```rust
use std::sync::atomic::{AtomicUsize, Ordering};
use std::thread;

fn main() {
    // 原子计数器
    let counter = AtomicUsize::new(0);
    let mut handlers = vec![];

    for _ in 0..10 {
        let handler = thread::spawn({
            let counter = &counter;
            move || {
                // 以原子方式累计，不会引发数据竞争
                // Ordering::SeqCst 确保所有线程对这个操作都有一致的视图
                counter.fetch_add(1, Ordering::SeqCst);
            }
        });
        handlers.push(handler);
    }

    for handler in handlers {
        handler.join().unwrap();
    }

    println!("Result: {}", counter.load(Ordering::SeqCst));
}
```



# 3. 读写锁

`std::sync::RWLock` 允许多读单写

```rust
use std::sync::{Arc, RwLock};
use std::thread;

fn main() {
    let lock = Arc::new(RwLock::new(5));
    let mut handlers = vec![];

    // 多读
    for _ in 0..10 {
        let lock = Arc::clone(&lock);
        let handler = thread::spawn(move || {
            // 支持并发读
            let r = lock.read().unwrap();
            println!("Read: {}", *r);
        });
        handlers.push(handler);
    }

    // 单写
    {
        let lock = Arc::clone(&lock);
        let handler = thread::spawn(move || {
            // 写操作，将阻塞其他读和写
            let mut w = lock.write().unwrap();
            *w += 1;
            println!("Write: {}", *w);
        });
        handlers.push(handler);
    }

    for handler in handlers {
        handler.join().unwrap();
    }
}
```



# 4. 条件变量

`std::sync::Condvar` 与 `Mutex` 一起使用，允许在线程之间执行更加复杂的同步操作。

```rust
use std::sync::{Arc, Condvar, Mutex};
use std::thread;

fn main() {
    // 条件变量用于协调两个线程，让一个线程等待另一个线程的信号
    let pair = Arc::new((Mutex::new(false), Condvar::new()));
    let pair2 = pair.clone();

    thread::spawn(move || {
        let (lock, cvar) = &*pair2;
        let mut started = lock.lock().unwrap();
        *started = true;
        cvar.notify_one();
    });

    let (lock, cvar) = &*pair;
    let mut started = lock.lock().unwrap();
    while !*started {
        // 在获得信号之前，阻塞当前线程
        // 一旦被通知，线程会继续执行
        started = cvar.wait(started).unwrap();
    }

    println!("Thread started");
}
```
